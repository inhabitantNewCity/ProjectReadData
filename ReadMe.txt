Для запуска программы необходимо иметь системную плату, которая подключается посредсством COM-port
конфигурация платы:
www.st.com/stm3f3discovery

Также в программе использовалась windows sdk10 и скомпилирована для WIN32
параметры компьютера:
Windows 10 Pro x64
Intel(R) Pentium(R) CPU B950 2.10GHz
.NET Framework 4.6.1

Для корректной работы программы так же необхордимо что бы путь к запускаемому файлу и базе данных (./CORP1FLOOR2.DPF) не были длинее 20 символов.

Для определения оптимального пути необходимо существование файла (<имя_базы>.g) гафа перемещений.

Для оптимальной работы также необходимо иметь две карты напряженности для wifi сигнала и геомагнитного излучения
